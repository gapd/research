PURPOSE OF THIS README FILE

This README provides pointers to crucial information about this
release of VisualWorks.


DISTRIBUTION CONTENTS

Refer to the /vw7.8/filelist.txt file for a complete listing of
files included with this distribution.

Some non-commercial distributions, such as CDs included with
magazines, may lack some files due to size constraints.  Any
missing files can be downloaded from:

     http://www.cincomsmalltalk.com

Then follow the "Download" link.


DOCUMENTATION

VisualWorks documentation is provided with this release in a set of
PDF files.  For a catalog of these documents, open /vw7.6/Welcome.pdf
in an Adobe Acrobat Reader.

A Full-text search index on these documents is provided for Adobe Readers
that support the feature.  Due to changes in the index file format, you
need Adobe AcroReader 7 to use these files. The files doc/index.pdx and 
all those under doc/index/ comprise this index, and may be deleted on 
systems lacking support for the feature.  You may need to add the index 
file to your reader's index list.  Select Edit -> Search -> Select Indexes, 
and add doc/index.pdx.

Installation instructions for this release are provided in the
Cincom Smalltalk Installation Guide (/vw7.6/Install.pdf or /vw7.6/Install.txt).

Additions, deletions, and special notes about this release are
provided in the VisualWorks 7.6 Release Notes (doc/ReleaseNotes7.6.pdf).
Release notes for previous releases are included as well in separate pdfs.

Additional information is provided in various readme.txt files.  For
example, there is a readme.txt file in each of the /vw7.6/bin/
subdirectories, containing specific information on the virtual
machines for each platform.
